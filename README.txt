──────────────────────────────────────────────────────────────
  LAS 5 COSAS QUE CAMBIAR PRIMERO
──────────────────────────────────────────────────────────────

  1. NOMBRE DEL RESTAURANTE
     Busca el nombre de ejemplo (Sol Café, Noche, FUEGO)
     y sustitúyelo por el tuyo. Aparece tanto en el HTML
     como en el objeto de traducciones del JavaScript
     (busca la palabra translations en el código).

  2. DIRECCIÓN Y DATOS DE CONTACTO
     Busca la dirección de ejemplo y escribe la tuya.
     Haz lo mismo con el teléfono y el correo electrónico.

  3. MENÚ Y PRECIOS
     El menú está dentro del objeto translations en el
     JavaScript. Busca la palabra menu y actualiza platos
     y precios para ES e EN.

  4. HORARIO DE APERTURA
     Busca el horario de ejemplo y actualiza con el tuyo.

  5. MAPA DE GOOGLE
     Ve a maps.google.com, busca tu dirección, haz clic en
     Compartir y luego en Insertar un mapa. Copia el código
     iframe. En el HTML, busca la etiqueta iframe en la
     sección del mapa y sustitúyela por la tuya.


──────────────────────────────────────────────────────────────
  SELECTOR DE IDIOMA (ES / EN)
──────────────────────────────────────────────────────────────

  Todas las plantillas tienen un selector de idioma en
  la barra de navegación con dos botones: ES | EN

  Por defecto el idioma es ESPAÑOL.
  El usuario puede cambiar al inglés con un clic.
  La elección se guarda en el navegador.

  Para añadir o modificar traducciones, busca el objeto
  const translations = { es: {...}, en: {...} } en el
  JavaScript y actualiza los valores para ambos idiomas.


──────────────────────────────────────────────────────────────
  MODO OSCURO
──────────────────────────────────────────────────────────────

  Cada plantilla tiene un botón de modo oscuro/claro en
  la barra de navegación. La preferencia del usuario
  se guarda automáticamente en el navegador.

  Sol Café y FUEGO arrancan en modo claro.
  Noche Fine Dining arranca en modo oscuro.


──────────────────────────────────────────────────────────────
  COLORES
──────────────────────────────────────────────────────────────

  Los colores principales están definidos como variables CSS
  al principio del archivo, en la sección :root { ... }.

  Para cambiar el color principal, encuentra la variable
  correspondiente y escribe tu color en formato HEX.

  Ejemplo:
    --red: #C1121F;    cambiar por tu color principal
    --yellow: #F4A261; cambiar por tu color secundario


──────────────────────────────────────────────────────────────
  FOTOS
──────────────────────────────────────────────────────────────

  Las plantillas usan fotos de ejemplo. Para usar las tuyas:

  OPCIÓN A (recomendada): Sube tus fotos a Cloudinary
    1. Crea una cuenta gratuita en cloudinary.com
    2. Sube tus fotos
    3. Copia la URL y pégala donde veas enlaces de ejemplo
       en el código HTML.

  OPCIÓN B: Fotos en local
    Pon tus fotos en la misma carpeta que el HTML
    y usa: <img src="tu-foto.jpg" ...>
