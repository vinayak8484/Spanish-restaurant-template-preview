  GETTING STARTED
──────────────────────────────────────────────────────────────

  1. Open the HTML file in your browser. It works immediately,
     no installation needed.

  2. To edit it, open the file in any text editor
     (Notepad, TextEdit, VS Code, etc.)

  3. Use CTRL+F (or CMD+F on Mac) to find the text
     you want to change.

  4. Save the file and refresh the browser to see changes.


──────────────────────────────────────────────────────────────
  THE 5 THINGS TO CHANGE FIRST
──────────────────────────────────────────────────────────────

  1. RESTAURANT NAME
     Search for the placeholder name (Sol Café, Noche, FUEGO)
     and replace it with yours. It appears in both the HTML
     and in the JavaScript translations object
     (search for the word translations in the code).

  2. ADDRESS AND CONTACT DETAILS
     Find the placeholder address and replace it with yours.
     Do the same for the phone number and email.

  3. MENU AND PRICES
     The menu is inside the translations object in the
     JavaScript. Search for menu and update dishes and prices
     for both ES and EN.

  4. OPENING HOURS
     Find the placeholder hours and update with yours.

  5. GOOGLE MAPS
     Go to maps.google.com, search your address, click Share
     and then Embed a map. Copy the iframe code.
     In the HTML, find the iframe tag in the map section
     and replace it with yours.


──────────────────────────────────────────────────────────────
  LANGUAGE SWITCHER (ES / EN)
──────────────────────────────────────────────────────────────

  All templates have a language selector in the navbar
  with two pill buttons: ES | EN

  Default language is SPANISH.
  Users can switch to English with one click.
  The choice is saved in the browser.

  To add or edit translations, find the object
  const translations = { es: {...}, en: {...} } in the
  JavaScript and update the values for both languages.


──────────────────────────────────────────────────────────────
  DARK MODE
──────────────────────────────────────────────────────────────

  Every template has a dark/light mode button in the navbar.
  The user's preference is saved automatically in the browser.

  Sol Café and FUEGO default to light mode.
  Noche Fine Dining defaults to dark mode.


──────────────────────────────────────────────────────────────
  COLOURS
──────────────────────────────────────────────────────────────

  The main colours are defined as CSS variables at the top
  of the file, inside the :root { ... } block.

  To change the primary colour, find the relevant variable
  and enter your colour as a HEX value.

  Example:
    --red: #C1121F;    swap for your main colour
    --yellow: #F4A261; swap for your accent colour


──────────────────────────────────────────────────────────────
  PHOTOS
──────────────────────────────────────────────────────────────

  The templates use placeholder photos. To use your own:

  OPTION A (recommended): Upload to Cloudinary (free)
    1. Create a free account at cloudinary.com
    2. Upload your photos
    3. Copy the URL and paste it in place of the example
       image links in the HTML.

  OPTION B: Local photos
    Put your photos in the same folder as the HTML file
    and use: <img src="your-photo.jpg" ...>
